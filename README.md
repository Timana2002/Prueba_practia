# Prueba_practia

Buenas tardes, cordial saludo.

Para la realizar la configuración, se debe aplicar lo siguiente:

1) Subir la base de datos, a un motor de base de datos, en este caso se uso un servidor local con MYSQL versión 5.7.31.
2) Se subir la aplicación a un servidor local o hosting y realizar la configuración que está en la dirección PRUEBA_PRACTICA/Connection/conexion.php. Solamente es cambiar las credenciales del servidor.
3) Ya estaria instalado el aplicativo.

Se usó PDO para la conexión y MVC.
